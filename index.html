⚙️ One-time Firebase setup needed

  1. Go to console.firebase.google.com and create a project
  2. Click Firestore Database → Create database → Start in test mode
  3. Go to Project Settings (gear icon) → scroll to "Your apps" → Add a Web app
  4. Copy the firebaseConfig object and paste it into the FIREBASE CONFIG section near the bottom of this file
  5. Redeploy to GitHub Pages — data will save automatically from then on
